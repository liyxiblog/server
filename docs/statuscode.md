# 状态码

## 用户状态

-2 - 被封禁

-1 - 被删除

0 - 未知

1 - 正常

## 文章状态

-3 - 违规文章

-2 - 审核不通过

0 - 未知

1 - 正常

2 - 待审核

特殊状态

10 - 隐藏文章（不在列表显示，但是可以被查看）



## http状态码

### 200 类型

正常 / 操作成功

200 -成功

2001 - 登录成功，勾选了保持登录状态

2002 - 登录成功，没有勾选保持登录状态

2011 - 项目列表请求成功（按照旧到新排序）

2012 - 项目列表请求成功（新的项目排在最前面）

### 400 类型

无法解析

400 - 客户端传递参数不足

4003 - token验证失败

4013 - 账号或密码错误

4093 - 验证码验证不通过

### 500 类型

服务器错误

5010 - 验证码接口失效
